定义

赖继娟 2024-04-25 13:52:07
Categories： Tags：

组件

组件是可复用的 Vue 实例，且带有一个名字，它们与 new Vue 接收相同的选项，例如 data、computed等

vue应用

通过 new Vue 创建的根 Vue 实例，以及可选的嵌套的、可复用的组件树组成；

指令

指令带有前缀 v-，以表示它们是 Vue 提供的特殊 attribute。当表达式的值改变时，将其产生的连带影响，响应式地作用于 DOM

声明式渲染（文本）

允许使用简洁的模版语法来将数据渲染进DOM系统

MVVM模式

是Model-View-ViewModel的简写，model是数据，view是用户界面，ViewModel可以取出 Model 的数据同时帮忙处理 View 中由于需要展示内容而涉及的业务逻辑。ViewModel将视图 UI 和业务逻辑分开，把Model和View关联起来，并实现双向数据绑定
MVVM核心思想，是关注model的变化，让MVVM框架利用自己的机制自动更新DOM

1.事件修饰符对事件进行约束等
